# Typing Speed Test

A responsive, interactive Typing Speed Test game built with HTML, CSS, and JavaScript.

## Features
- Measures typing speed and score
- Fun background music
- Correct and wrong sound effects
- Confetti animation on correct words
- Responsive layout for mobile and desktop
- High score tracking with localStorage

## How to Play
1. Click **Start Game**
2. Type the word displayed in the input box
3. Correct words increase your score and trigger confetti
4. Incorrect typing plays a wrong sound
5. The game ends when the timer reaches 0

## Technologies used
1. HTML5 – for the structure and semantic layout of the Typing Speed Test game.
2. CSS3 – for styling, responsiveness, and visual effects (including background and animations).
3. JavaScript (ES6) – for game logic, interactivity, scoring, and event handling.
4. Audio API (HTML <audio> tag) – for background music and sound effects (correct/wrong).
5. Confetti Animation (JavaScript) – to create fun visual feedback for correct answers.
6. Git & GitHub – for version control and repository hosting.
7. GitHub Pages – to deploy and share the live playable version of the game in browsers.

## How to Run
1. Open `index.html` in a browser, or
2. Deploy via GitHub Pages
